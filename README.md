# 🧠 QuizMaster — Interactive Web Testing System

Повноцінний веб-застосунок для створення та проходження квізів. Реалізовано з акцентом на семантичну верстку та сучасний JavaScript.

## 🚀 Основні можливості:
* **Два режими тестування:** Класичний (спокійний) та Kahoot-style (на час).
* **Панель адміністратора:** Додавання, редагування та видалення запитань у реальному часі.
* **Авторизація:** Система входу та реєстрації користувачів.
* **Результати:** Детальна статистика після проходження (відсоток успіху, кількість помилок).

## 💻 Технологічний стек:
* **Frontend:** HTML5 (семантична розмітка), CSS3, JavaScript (ES6+).
* **Backend:** Node.js, Express.
* **Database:** SQLite (для зберігання запитань та користувачів).

## 🛠️ Як запустити локально:
1. `npm install` — встановлення залежностей.
2. `node server.js` — запуск сервера.
3. Відкрити `http://localhost:3000` у браузері.
